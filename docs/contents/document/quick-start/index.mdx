# Quick Start

This page gives you the shortest path to get started. For details, jump to the subpages.

## 1) Check requirements

- See: [Requirements](./requirements)

## 2) Install and enable

```bash
pip install abe-kafka
export MESSAGE_QUEUE_BACKEND=kafka
export KAFKA_BOOTSTRAP_SERVERS=localhost:9092
```

For SASL/SSL examples and more options, see: [Installation](./installation)

## 3) Validate locally

- Run the minimal publish/consume example here: [How to Run](./how-to-run)
